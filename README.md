# 🎓 Student Performance Predictor

A Machine Learning web application that predicts a student's **exam score** based on study habits and background factors such as hours studied, attendance, parental involvement, distance from home, and gender.

Built using **Python, Scikit-learn, and Streamlit**.

---

## 🚀 Features

- Predict student exam scores using a trained **Random Forest model**
- Interactive **Streamlit web interface**
- Easy input controls using sliders and dropdowns
- Real-time predictions

---

## 🧠 Machine Learning Model

The model is trained using **RandomForestRegressor** from Scikit-learn.

### Input Features
- Hours Studied
- Attendance
- Parental Involvement
- Distance from Home
- Gender

### Output
- Predicted Exam Score
